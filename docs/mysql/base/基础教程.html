                                                                                                        # ж•°жҚ®иҜӯеҸҘж“ҚдҪңзұ»еһӢ

                                                                                                        SQLжҳҜж“ҚдҪңж•°жҚ®еә“зҡ„ж ёеҝғпјҢ

                                                                                                        з»“жһ„еҢ–жҹҘиҜўиҜӯиЁҖ(Structured Query Language)з®Җз§°SQLпјҢжҳҜдёҖз§Қзү№ж®Ҡзӣ®зҡ„зҡ„зј–зЁӢиҜӯиЁҖпјҢжҳҜдёҖз§Қж•°жҚ®еә“жҹҘиҜўе’ҢзЁӢеәҸи®ҫи®ЎиҜӯиЁҖпјҢз”ЁдәҺеӯҳеҸ–ж•°жҚ®д»ҘеҸҠжҹҘиҜўгҖҒжӣҙж–°е’Ңз®ЎзҗҶе…ізі»ж•°жҚ®еә“зі»з»ҹпјӣеҗҢж—¶д№ҹжҳҜж•°жҚ®еә“и„ҡжң¬ж–Үд»¶зҡ„жү©еұ•еҗҚгҖӮ

                                                                                                        SQLжҳҜжңҖйҮҚиҰҒзҡ„е…ізі»ж•°жҚ®еә“ж“ҚдҪңиҜӯиЁҖпјҢе№¶дё”е®ғзҡ„еҪұе“Қе·Із»Ҹи¶…еҮәж•°жҚ®еә“йўҶеҹҹпјҢеҫ—еҲ°е…¶д»–йўҶеҹҹзҡ„йҮҚи§Ҷе’ҢйҮҮз”ЁпјҢеҰӮдәәе·ҘжҷәиғҪйўҶеҹҹзҡ„ж•°жҚ®жЈҖзҙўзӯүгҖӮ

                                                                                                        SQLжҳҜе…ізі»жЁЎеһӢзҡ„ж•°жҚ®еә“еә”з”ЁиҜӯиЁҖпјҢз”ұIBMеңЁ20дё–зәӘ70е№ҙд»Јдёәе…¶е…ізі»еһӢж•°жҚ®еә“ System R жүҖејҖеҸ‘гҖӮ

                                                                                                        SQL жҳҜ1986е№ҙ10 жңҲз”ұзҫҺеӣҪеӣҪе®¶ж ҮеҮҶеұҖпјҲANSIпјүйҖҡиҝҮзҡ„ж•°жҚ®еә“иҜӯиЁҖзҫҺеӣҪж ҮеҮҶпјҢжҺҘзқҖпјҢеӣҪйҷ…ж ҮеҮҶеҢ–з»„з»ҮпјҲISOпјүйўҒеёғдәҶSQLжӯЈејҸеӣҪйҷ…ж ҮеҮҶгҖӮ1989е№ҙ4жңҲпјҢISOжҸҗеҮәдәҶе…·жңүе®Ңж•ҙжҖ§зү№еҫҒзҡ„SQL89ж ҮеҮҶпјҢ1992е№ҙ11жңҲеҸҲе…¬еёғдәҶSQL92ж ҮеҮҶгҖӮ

                                                                                                        иҷҪз„¶еҗ„дёӘж•°жҚ®еә“зі»з»ҹз•ҘжңүдёҚеҗҢпјҢдҪҶжҳҜд»–д»¬еҹәжң¬еқҮйҒөеҫӘSQL 92ж ҮеҮҶгҖӮжҲ–иҖ…еңЁSQL 92дёҠеҒҡдәҶдёҖдәӣз®ҖеҚ•зҡ„жү©еұ•е’ҢеҸҳеҢ–гҖӮ

                                                                                                        еӯҰеҘҪдәҶMySQL зҡ„SQL иҜӯжі•пјҢе…¶д»–зҡ„SQLиҜӯжі•еӯҰд№ иө·жқҘеқҮжҳҜдёҮеҸҳдёҚзҰ»е…¶дёӯгҖӮ

                                                                                                        1. ж•°жҚ®е®ҡд№үиҜӯиЁҖ(DDL пјҢData Defintion Language)иҜӯеҸҘпјҡж•°жҚ®е®ҡд№үиҜӯеҸҘпјҢз”ЁдәҺе®ҡд№үдёҚеҗҢзҡ„ж•°жҚ®ж®өгҖҒж•°жҚ®еә“гҖҒиЎЁгҖҒеҲ—гҖҒзҙўеј•зӯүгҖӮеёёз”Ёзҡ„иҜӯеҸҘе…ій”®еӯ—еҢ…жӢ¬createгҖҒdropгҖҒalterзӯүгҖӮ
                                                                                                        2. ж•°жҚ®ж“ҚдҪңиҜӯиЁҖ(DML пјҢ Data Manipulation Language)иҜӯеҸҘпјҡж•°жҚ®ж“ҚзәөиҜӯеҸҘпјҢз”ЁдәҺж·»еҠ гҖҒеҲ йҷӨгҖҒжӣҙж–°е’ҢжҹҘиҜўж•°жҚ®еә“и®°еҪ•пјҢе№¶жЈҖжҹҘж•°жҚ®зҡ„е®Ңж•ҙжҖ§гҖӮеёёз”Ёзҡ„иҜӯеҸҘе…ій”®еӯ—дё»иҰҒеҢ…жӢ¬insertгҖҒdeleteгҖҒupdateе’ҢselectзӯүгҖӮ
                                                                                                        3. ж•°жҚ®жҺ§еҲ¶иҜӯиЁҖ(DCLпјҢ Data Control Language)иҜӯеҸҘпјҡж•°жҚ®жҺ§еҲ¶иҜӯеҸҘпјҢз”ЁдәҺжҺ§еҲ¶дёҚеҗҢж•°жҚ®ж®өзӣҙжҺҘзҡ„и®ёеҸҜе’Ңи®ҝй—®зә§еҲ«зҡ„иҜӯеҸҘгҖӮиҝҷдәӣиҜӯеҸҘе®ҡд№үдәҶж•°жҚ®еә“гҖҒиЎЁгҖҒеӯ—ж®өгҖҒз”ЁжҲ·зҡ„и®ҝй—®жқғйҷҗе’Ңе®үе…Ёзә§еҲ«гҖӮдё»иҰҒзҡ„иҜӯеҸҘе…ій”®еӯ—еҢ…жӢ¬grantгҖҒrevokeзӯүгҖӮ

                                                                                                        blob0-65 535 еӯ—иҠӮдәҢиҝӣеҲ¶еҪўејҸзҡ„й•ҝж–Үжң¬ж•°жҚ®

                                                                                                        mediumtext0-16 777 215 еӯ—иҠӮдёӯзӯүй•ҝеәҰж–Үжң¬ж•°жҚ®

                                                                                                        longtext0-4 294 967 295 еӯ—иҠӮжһҒеӨ§ж–Үжң¬ж•°жҚ®

                                                                                                        • ж•ҲзҺҮй«ҳпјҡеӯҳеӮЁиҝҮзЁӢзј–иҜ‘дёҖж¬ЎеҗҺпјҢе°ұдјҡеӯҳеҲ°ж•°жҚ®еә“пјҢжҜҸж¬Ўи°ғз”Ёж—¶йғҪзӣҙжҺҘжү§иЎҢгҖӮиҖҢжҷ®йҖҡзҡ„sqlиҜӯеҸҘжҲ‘д»¬иҰҒдҝқеӯҳеҲ°е…¶д»–ең°ж–№пјҲдҫӢеҰӮпјҡи®°дәӢжң¬ дёҠпјүпјҢйғҪиҰҒе…ҲеҲҶжһҗзј–иҜ‘жүҚдјҡжү§иЎҢгҖӮжүҖд»ҘжғіеҜ№иҖҢиЁҖеӯҳеӮЁиҝҮзЁӢж•ҲзҺҮжӣҙй«ҳгҖӮ
                                                                                                        • йҷҚдҪҺзҪ‘з»ңжөҒйҮҸпјҡеӯҳеӮЁиҝҮзЁӢзј–иҜ‘еҘҪдјҡж”ҫеңЁж•°жҚ®еә“пјҢжҲ‘д»¬еңЁиҝңзЁӢи°ғз”Ёж—¶пјҢдёҚдјҡдј иҫ“еӨ§йҮҸзҡ„еӯ—з¬ҰдёІзұ»еһӢзҡ„sqlиҜӯеҸҘгҖӮ
                                                                                                        • еӨҚз”ЁжҖ§й«ҳпјҡеӯҳеӮЁиҝҮзЁӢеҫҖеҫҖжҳҜй’ҲеҜ№дёҖдёӘзү№е®ҡзҡ„еҠҹиғҪзј–еҶҷзҡ„пјҢеҪ“еҶҚйңҖиҰҒе®ҢжҲҗиҝҷдёӘзү№е®ҡзҡ„еҠҹиғҪж—¶пјҢеҸҜд»ҘеҶҚж¬Ўи°ғз”ЁиҜҘеӯҳеӮЁиҝҮзЁӢгҖӮ
                                                                                                        • еҸҜз»ҙжҠӨжҖ§й«ҳпјҡеҪ“еҠҹиғҪиҰҒжұӮеҸ‘з”ҹе°Ҹзҡ„еҸҳеҢ–ж—¶пјҢдҝ®ж”№д№ӢеүҚзҡ„еӯҳеӮЁиҝҮзЁӢжҜ”иҫғе®№жҳ“пјҢиҠұиҙ№зІҫеҠӣе°‘гҖӮ
                                                                                                        • е®үе…ЁжҖ§й«ҳпјҡе®ҢжҲҗжҹҗдёӘзү№е®ҡеҠҹиғҪзҡ„еӯҳеӮЁиҝҮзЁӢдёҖиҲ¬еҸӘжңүзү№е®ҡзҡ„з”ЁжҲ·еҸҜд»ҘдҪҝз”ЁпјҢе…·жңүдҪҝз”Ёиә«д»ҪйҷҗеҲ¶пјҢжӣҙе®үе…ЁгҖӮ

                                                                                                        • дәӢеҠЎеӨ„зҗҶеҸҜд»Ҙз”ЁжқҘз»ҙжҠӨж•°жҚ®еә“зҡ„е®Ңж•ҙжҖ§пјҢдҝқиҜҒжҲҗжү№зҡ„ sql иҜӯеҸҘиҰҒд№Ҳе…ЁйғЁжү§иЎҢпјҢиҰҒд№Ҳе…ЁйғЁдёҚжү§иЎҢгҖӮ
                                                                                                        • дҝ®ж”№ж•°жҚ®зҡ„е‘Ҫд»ӨдјҡиҮӘеҠЁзҡ„и§ҰеҸ‘дәӢеҠЎпјҢеҢ…жӢ¬insertгҖҒupdateгҖҒdelete

                                                                                                        дёҖиҲ¬жқҘиҜҙпјҢдәӢеҠЎжҳҜеҝ…йЎ»ж»Ўи¶і4дёӘжқЎд»¶пјҲacidпјүпјҡпјҡеҺҹеӯҗжҖ§пјҲatomicityпјҢжҲ–з§°дёҚеҸҜеҲҶеүІжҖ§пјүгҖҒдёҖиҮҙжҖ§пјҲconsistencyпјүгҖҒйҡ”зҰ»жҖ§пјҲisolationпјҢеҸҲз§°зӢ¬з«ӢжҖ§пјүгҖҒжҢҒд№…жҖ§пјҲdurabilityпјүгҖӮ

                                                                                                        • еҺҹеӯҗжҖ§пјҡдёҖдёӘдәӢеҠЎпјҲtransactionпјүдёӯзҡ„жүҖжңүж“ҚдҪңпјҢиҰҒд№Ҳе…ЁйғЁе®ҢжҲҗпјҢиҰҒд№Ҳе…ЁйғЁдёҚе®ҢжҲҗпјҢдёҚдјҡз»“жқҹеңЁдёӯй—ҙжҹҗдёӘзҺҜиҠӮгҖӮдәӢеҠЎеңЁжү§иЎҢиҝҮзЁӢдёӯеҸ‘з”ҹй”ҷиҜҜпјҢдјҡиў«еӣһж»ҡпјҲrollbackпјүеҲ°дәӢеҠЎејҖе§ӢеүҚзҡ„зҠ¶жҖҒпјҢе°ұеғҸиҝҷдёӘдәӢеҠЎд»ҺжқҘжІЎжңүжү§иЎҢиҝҮдёҖж ·гҖӮ
                                                                                                        • дёҖиҮҙжҖ§пјҡеңЁдәӢеҠЎејҖе§Ӣд№ӢеүҚе’ҢдәӢеҠЎз»“жқҹд»ҘеҗҺпјҢж•°жҚ®еә“зҡ„е®Ңж•ҙжҖ§жІЎжңүиў«з ҙеқҸгҖӮиҝҷиЎЁзӨәеҶҷе…Ҙзҡ„иө„ж–ҷеҝ…йЎ»е®Ңе…Ёз¬ҰеҗҲжүҖжңүзҡ„йў„и®ҫи§„еҲҷпјҢиҝҷеҢ…еҗ«иө„ж–ҷзҡ„зІҫзЎ®еәҰгҖҒдёІиҒ”жҖ§д»ҘеҸҠеҗҺз»ӯж•°жҚ®еә“еҸҜд»ҘиҮӘеҸ‘жҖ§ең°е®ҢжҲҗйў„е®ҡзҡ„е·ҘдҪңгҖӮ
                                                                                                        • йҡ”зҰ»жҖ§пјҡж•°жҚ®еә“е…Ғи®ёеӨҡдёӘе№¶еҸ‘дәӢеҠЎеҗҢж—¶еҜ№е…¶ж•°жҚ®иҝӣиЎҢиҜ»еҶҷе’Ңдҝ®ж”№зҡ„иғҪеҠӣпјҢйҡ”зҰ»жҖ§еҸҜд»ҘйҳІжӯўеӨҡдёӘдәӢеҠЎе№¶еҸ‘жү§иЎҢж—¶з”ұдәҺдәӨеҸүжү§иЎҢиҖҢеҜјиҮҙж•°жҚ®зҡ„дёҚдёҖиҮҙгҖӮдәӢеҠЎйҡ”зҰ»еҲҶдёәдёҚеҗҢзә§еҲ«пјҢеҢ…жӢ¬иҜ»жңӘжҸҗдәӨпјҲread uncommittedпјүгҖҒиҜ»жҸҗдәӨпјҲread committedпјүгҖҒеҸҜйҮҚеӨҚиҜ»пјҲrepeatable readпјүе’ҢдёІиЎҢеҢ–пјҲserializableпјүгҖӮ
                                                                                                        • жҢҒд№…жҖ§пјҡдәӢеҠЎеӨ„зҗҶз»“жқҹеҗҺпјҢеҜ№ж•°жҚ®зҡ„дҝ®ж”№е°ұжҳҜж°ёд№…зҡ„пјҢеҚідҫҝзі»з»ҹж•…йҡңд№ҹдёҚдјҡдёўеӨұгҖӮ

                                                                                                        • rollback д№ҹеҸҜд»ҘдҪҝз”Ё rollback workпјҢдёҚиҝҮдәҢиҖ…жҳҜзӯүд»·зҡ„гҖӮеӣһж»ҡдјҡз»“жқҹз”ЁжҲ·зҡ„дәӢеҠЎпјҢе№¶ж’Өй”ҖжӯЈеңЁиҝӣиЎҢзҡ„жүҖжңүжңӘжҸҗдәӨзҡ„дҝ®ж”№пјӣ

                                                                                                        • savepoint identifierпјҢsavepoint е…Ғи®ёеңЁдәӢеҠЎдёӯеҲӣе»әдёҖдёӘдҝқеӯҳзӮ№пјҢдёҖдёӘдәӢеҠЎдёӯеҸҜд»ҘжңүеӨҡдёӘ savepointпјӣ

                                                                                                        • release savepoint identifier еҲ йҷӨдёҖдёӘдәӢеҠЎзҡ„дҝқеӯҳзӮ№пјҢеҪ“жІЎжңүжҢҮе®ҡзҡ„дҝқеӯҳзӮ№ж—¶пјҢжү§иЎҢиҜҘиҜӯеҸҘдјҡжҠӣеҮәдёҖдёӘејӮеёёпјӣ

                                                                                                        • rollback to identifier жҠҠдәӢеҠЎеӣһж»ҡеҲ°ж Үи®°зӮ№пјӣ

                                                                                                        • set transaction з”ЁжқҘи®ҫзҪ®дәӢеҠЎзҡ„йҡ”зҰ»зә§еҲ«гҖӮ innodb еӯҳеӮЁеј•ж“ҺжҸҗдҫӣдәӢеҠЎзҡ„йҡ”зҰ»зә§еҲ«жңү:
