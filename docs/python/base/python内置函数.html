                                      sorted 和 sort 区别

                                      • sorted函数对所有可迭代对象都有效，结果是一个新的列表，也就是原对象没有改变
                                      • sort 方法是 list 独有的方法，是对列表本身进行排序，也就是原列表进行了改变
